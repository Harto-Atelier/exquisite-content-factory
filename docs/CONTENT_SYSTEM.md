# Content System

## Daily target

30 videos/day:

- 15 story / brand / worldbuilding
- 15 utility / website / onboarding

This should be generated from repeatable formats, not handwritten from scratch every day.

## Production tiers

### Tier 1: Hero videos
1-3/day. Higher-quality worldbuilding, avatar, custom b-roll, stronger sound.

### Tier 2: Template videos
10-15/day. Generated from stable scripts and reusable visual templates.

### Tier 3: Micro-variations
10-15/day. Trend hooks, caption-led videos, screen snippets, fast utility.

## Trend-jacking rule

Copy the structure, not the identity.

Allowed:
- hook pattern
- pacing
- edit rhythm
- caption rhythm
- POV/meme format
- narrative twist

Do not copy:
- protected characters
- exact visual identity
- exact script
- logos/music/footage without rights

Translate trends into Exquisite:

- POV: you open the wrong door → POV: Noura opens the Claim Kiosk.
- Nobody tells you this about... → Nobody tells you this about wallet galleries.
- Things that feel illegal but aren't → Things in Exquisite World that feel impossible but are on-chain.
- Wait until the end → Wait until Noura chooses the artwork.

## Script shape

1. Hook: simple relatable or curiosity line.
2. Human moment: Lucía/Kamau emotion.
3. Noura action: comic/magic discovery.
4. Curator note: artist/curator meaning.
5. CTA: soft website/world ritual.
