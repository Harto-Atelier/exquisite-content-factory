# Exquisite World Lore Repository

This folder is the public canon source for Exquisite World.

It is designed to be readable by humans and usable by other repositories, websites, agents, and content tools.

## What lives here

- Long-term story bible for the next 10 years.
- Character records for Lucía, Kamau, Noura, Agents, Curators, and future cast.
- World registry: locations, objects, rituals, visual motifs, website metaphors.
- Timeline: seasons, arcs, major milestones.
- Episode system: repeatable story formats for videos, comics, pages, games, and posts.
- Style guides: tone, visual language, Pixar-level emotional simplicity without copying protected IP.
- Machine-readable JSON data for the main website/app.

## Canon rule

If it is not recorded here, it is not canon yet.

New story ideas can start as drafts, but once they affect characters, world rules, website copy, or long-term continuity, they must be added to the registry.

## Public repo rule

This repo can be public. Do not commit secrets, private launch operations, API keys, unreleased wallet data, or sensitive internal metrics.

Public readers should feel like they are entering a living art-world archive.
Developers should be able to import structured data from `lore/07-website-data/`.
